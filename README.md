# Interactive Excel Sales Dashboard

## Project Overview
This project is an interactive Excel Sales Dashboard created using Microsoft Excel.  
The dashboard helps analyze sales performance using Pivot Tables, Charts, Slicers, and VBA automation.



## Features
- Interactive Slicers
- Dynamic Charts
- KPI Cards
- Region-wise Sales Analysis
- Sales Executive Performance Tracking
- Automated Filtering using VBA
- User-friendly Dashboard Design



## Tools & Technologies Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- VBA Macros
